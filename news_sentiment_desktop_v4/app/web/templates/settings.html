Gmail 新聞來源

首次連接教學（Google Cloud Console 申請 OAuth 憑證，只需設定一次）
  1. 開啟 console.cloud.google.com，登入 Google 帳號，上方選單建立新專案
  2. 左側「API 和服務 → 程式庫」搜尋 Gmail API → 啟用
  3. 「OAuth 同意畫面」：使用者類型選「外部」；重要：在「測試使用者」加入收新聞監測信的 Gmail 地址（沒加會被「應用程式未經驗證」擋下）
  4. 「憑證 → 建立憑證 → OAuth 用戶端 ID」：應用程式類型選「網頁應用程式」，並把下方藍框顯示的 Authorized redirect URI 逐字貼進「已授權的重新導向 URI」
  5. 把產生的 Client ID / Client Secret 設為部署平台的環境變數 GMAIL_OAUTH_CLIENT_ID / GMAIL_OAUTH_CLIENT_SECRET（Render：Dashboard → Environment）並重新部署
  6. 回到本頁按「連接 Gmail」→ 登入收新聞信的帳號 → 出現「未經驗證」警告點「繼續」→ 允許唯讀權限

本系統只申請「唯讀」權限，只能讀信、不能寄信或刪信。
